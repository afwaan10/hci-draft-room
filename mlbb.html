HOK Draft Room by HCI

Draft Room by HCI is a draft simulation and training platform for coaches, analysts, and teams. Admin creates the room, the assigned host runs the draft, and every action is logged by the website.

HOK Room Access

Admin Room Setup

Rooms are created by admin. The assigned user enters the Room ID and Host Code to become host. Users do not receive Delete, Reset, or Leave controls.

Admin Create Room

Create an assigned draft room. Share the Room ID and Host Code only with the host.

Enter Room

Enter Room ID to spectate or join Team B. Enter Host Code only if the room was assigned to you as host.
